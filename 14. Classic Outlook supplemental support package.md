## Page 1 copy  
## Shared mailboxes in Classic Outlook  
## Before you start  
You need approved access to the mailbox.  
Access may include:  
* Read and manage  
* Send As  
* Send on Behalf  
A shared mailbox is accessed through your own work account. Do not sign in directly using the shared mailbox address.  
  
## Open the shared mailbox  
## If it appears automatically  
1. Restart Outlook after access is granted.  
2. Find the mailbox in the Folder Pane.  
3. Select the arrow beside its name.  
4. Open **Inbox**.  
## If it does not appear  
1. Select **File**.  
2. Select **Account Settings**.  
3. Select **Account Settings** again.  
4. Select your work account.  
5. Select **Change**.  
6. Select **More Settings**.  
7. Open **Advanced**.  
8. Select **Add**.  
9. Enter the shared mailbox.  
10. Complete the prompts and restart Outlook.  
**Get help:** [Insert shared mailbox access path]  
  
## Send from the shared mailbox  
1. Select **New Email**.  
2. Open **Options**.  
3. Select **From**.  
4. Select the shared mailbox address.  
5. Add recipients and the subject.  
6. Write the message.  
7. Confirm the **From** address.  
8. Review and select **Send**.  
If the mailbox is not listed:  
1. Select **From**.  
2. Select **Other Email Address**.  
3. Enter the shared mailbox address.  
  
## Reply from the shared mailbox  
1. Open the message in the shared mailbox.  
2. Select **Reply** or **Reply All**.  
3. Check the **From** field.  
4. Select the shared address if needed.  
5. Review the conversation.  
6. Confirm no one else has replied.  
7. Write and send the response.

## Manage shared work clearly  
## Before replying

2. Select the Search box.  
3. Confirm the search scope.

5. Add filters when needed.  
6. Confirm the result belongs to the shared mailbox.

1. Review the full conversation.  
2. Check shared Sent Items when available.  
3. Review flags, categories or team status.  
4. Confirm ownership.  
5. Update the shared status.  
6. Reply only when ownership is clear.

| Search misses messages | Confirm mailbox, folder and search scope |
| Messages do not refresh | Send/Receive, connection and Outlook restart |

## Start here  
Copilot in Classic Outlook requires:  
* Company work account  
* Microsoft 365 Copilot license  
* Supported Classic Outlook version  
* Primary Exchange Online mailbox  
* Required connected experiences  
* Supported message type  
Copilot Outlook features are not available inside shared, delegate, group or archive mailboxes.  
  
## Copilot button is missing  
## Check  
1. Are you using your company work account?  
2. Is the message in your primary mailbox?  
3. Are you in Classic Outlook?  
4. Is Outlook updated?  
5. Did you restart Outlook?  
6. Are connected experiences available?  
7. Does Copilot appear in another supported message?  
## Try  
* Close and reopen Outlook.  
* Open a new email in the primary mailbox.  
* Open a standard unencrypted email thread.  
* Check for approved Microsoft 365 updates.  
* Test Copilot in another Microsoft 365 application.  
## Get help when  
* Copilot is missing from every supported Outlook message  
* Your license is active but the command never appears  
* Copilot stopped appearing after an update  
  
## Copilot cannot summarize the thread  
## Check  
* The thread is in the primary mailbox.  
* It contains multiple messages.  
* The email is not in a shared mailbox.  
* The message does not use unsupported encryption.  
* Outlook is online.  
## Try  
1. Close and reopen the thread.  
2. Restart Outlook.  
3. Try a different low-risk thread.  
4. Compare behavior in New Outlook or Outlook on the web when permitted.  
  
## Draft with Copilot does not work  
## Check  
* Your prompt is clear.  
* The message is in a supported mailbox.  
* Outlook is connected.  
* The prompt does not request unsupported or missing information.  
## Improve the prompt  
Use:  
Draft a warm professional reply to this thread. Ask for the missing effective date, keep it under 100 words and do not add new facts.  
Avoid:  
Reply to this.  
  
## The draft contains incorrect information  
1. Do not send it.  
2. Compare it with the original email.  
3. Remove invented or unsupported details.  
4. Correct names, dates, owners and deadlines.  
5. Add information only you know.  
6. Complete the final review.  
  
## Coaching suggestions do not fit  
* Apply only the suggestions that help.  
* Keep required deadlines and direct requests.  
* Restore context that was removed.  
* Do not accept tone changes automatically.

## Asset 5 — Search syntax cheat sheet  
## Asset overview  
**Title:** Search faster in Classic Outlook **Subtitle:** Useful search terms and combinations **Format:** One-page cheat sheet **Audience:** All guide users **Estimated use time:** 30–90 seconds  
## Copy deck  
## Search faster in Classic Outlook  
## Before searching  
1. Select the Search box or press **Ctrl+E**.  
2. Check the search scope.  
3. Use the most specific detail you know.  
4. Combine search terms when needed.  
5. Clear the search when finished.  
## Common search terms  

| Need                         | Enter                      |
| ---------------------------- | -------------------------- |
| From a person                | from:"Jordan White"        |
| Sent to a person             | to:"Jordan White"          |
| Copied to a person           | cc:"Jordan White"          |
| Subject contains a phrase    | subject:"renewal proposal" |
| Exact phrase anywhere        | "effective date"           |
| Has an attachment            | hasattachments:yes         |
| Does not have an attachment  | hasattachments:no          |
| Is flagged                   | isflagged:yes              |
| Is unread                    | isread:no                  |
| Is read                      | isread:yes                 |
| Has a category               | category:"Waiting"         |
| Received today               | received:today             |
| Received yesterday           | received:yesterday         |
| Received this week           | received:this week         |
| Sent this week               | sent:this week             |
| Important message            | importance:high            |
| File name or extension       | attachment:xlsx            |
| Message body contains a word | body:renewal               |
  
****Useful combinations****  
## Find renewal messages from Jordan  
```
from:"Jordan White" renewal

```
## Find Jordan’s renewal messages with attachments  
```
from:"Jordan White" renewal hasattachments:yes

```
## Find unread messages categorized as Waiting  
```
isread:no category:"Waiting"

```
## Find flagged messages received this week  
```
isflagged:yes received:this week

```
## Find messages with Excel attachments  
```
hasattachments:yes attachment:xlsx

```
## Find an exact subject phrase  
```
subject:"Q3 renewal review"

```
## Search scope matters  

| Scope             | Searches                          |
| ----------------- | --------------------------------- |
| Current Folder    | Only the open folder              |
| Subfolders        | The folder and folders beneath it |
| Current Mailbox   | The selected mailbox              |
| All Mailboxes     | All available mailboxes           |
| All Outlook Items | Mail and other Outlook items      |
  
****Search a shared mailbox****  
1. Open a folder inside the shared mailbox.  
2. Select the Search box.  
3. Set the scope to the current folder or shared mailbox.  
4. Enter the search terms.  
5. Confirm the result location.  
## Search tips  
* Put quotation marks around exact phrases.  
* Add one filter at a time.  
* Try a sender or subject when a broad keyword returns too much.  
* Check spelling.  
* Check Archive, Deleted Items and shared mailbox folders.  
* Review rules when messages appear to be missing.  
## What to remember  
Search scope determines where Outlook looks. Search terms determine what Outlook returns.  
## Avoid this  
* Do not assume no result means the message was deleted.  
* Do not search All Mailboxes when one folder is enough.  
* Do not leave an old search active while reviewing new email.
