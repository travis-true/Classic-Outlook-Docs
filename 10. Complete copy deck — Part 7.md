## Page 64 — Resolve connection, search and rule issues  
## Outlook shows Working Offline or Disconnected  
## Check first  
1. Look at the Outlook status bar.  
2. Confirm that the Windows laptop has an internet connection.  
3. Open the **Send/Receive** tab.  
4. Check whether **Work Offline** is selected.  
5. If it is selected, select **Work Offline** again.  
6. Wait for Outlook to reconnect.  
7. Select **Send/Receive All Folders** when needed.  
## Try this  
* Close and reopen Outlook.  
* Reconnect to the approved network or VPN when required.  
* Confirm that other Microsoft 365 services are available.  
* Restart the laptop if Outlook remains disconnected.  
## Get help when  
* Outlook repeatedly disconnects  
* The status remains Working Offline  
* Messages remain in the Outbox  
* Other Microsoft 365 applications also cannot connect  
  
## Search results are incomplete  
## Check first  
1. Confirm the search term.  
2. Check the search scope.  
3. Confirm the correct mailbox and folder.  
4. Clear any filters.  
5. Check Archive and Deleted Items.  
6. Check whether a rule moved the message.  
7. Search by sender, subject or attachment instead of a broad keyword.  
## Try this  
* Search the current mailbox instead of the current folder.  
* Use fewer search terms.  
* Search for an exact phrase.  
* Restart Outlook.  
* Test whether the message appears in Outlook on the web.  
## Get help when  
* New messages never appear in search  
* Search repeatedly returns no results  
* Shared mailbox search behaves inconsistently  
* Outlook reports an indexing or search error  
  
## A rule does not run  
## Check first  
1. Open **File > Manage Rules & Alerts**.  
2. Confirm that the rule is turned on.  
3. Review its conditions.  
4. Review its exceptions.  
5. Confirm that the destination folder still exists.  
6. Review the order of overlapping rules.  
7. Confirm whether the rule is marked as client-only.  
## Try this  
1. Select **Run Rules Now**.  
2. Run the rule on a small test folder.  
3. Confirm which messages it processes.  
4. Adjust the rule.  
5. Test it again.  
## Get help when  
* Rules repeatedly stop working  
* A rule processes sensitive mail incorrectly  
* A client-only rule must run while Outlook is closed  
* The rule involves automatic forwarding or redirection

## Page 65 — Resolve shared mailbox and follow-up issues  
## A shared mailbox is missing  
## Check first  
1. Confirm that access was approved.  
2. Close and reopen Outlook.  
3. Expand the Folder Pane.  
4. Confirm that the correct Outlook profile is open.  
5. Check whether the mailbox was added manually.  
6. Wait for recently approved access to take effect.  
## Get help when  
* The mailbox remains unavailable  
* You receive a permission error  
* Only some folders appear  
* The mailbox works in one Outlook platform but not another  
**Support path:** [Insert shared mailbox support or access-request path]  
  
## The shared From address is unavailable  
## Check first  
1. Create a new message.  
2. Select **Options > From**.  
3. Open the From list.  
4. Choose **Other Email Address** when needed.  
5. Enter the shared mailbox address.  
6. Confirm that you have Send As or Send on Behalf permission.  
## Get help when  
* The address cannot be selected  
* Sending produces a permission error  
* The message sends from the wrong address  
* The team requires different sending permissions  
  
## A shared mailbox is not refreshing  
## Check first  
1. Select another folder, then return to the shared Inbox.  
2. Select **Send/Receive All Folders**.  
3. Check the status bar.  
4. Close and reopen Outlook.  
5. Compare the mailbox with Outlook on the web.  
## Get help when  
* New messages appear only after restarting Outlook  
* Outlook remains on “Updating this folder”  
* Only some shared folders refresh  
  
## A flag reminder did not appear  
## Check first  
1. Open the flagged message.  
2. Review the follow-up date.  
3. Confirm that the Reminder checkbox is selected.  
4. Confirm the reminder date and time.  
5. Check whether the reminder was dismissed.  
6. Confirm that Outlook was running and connected.  
## Try this  
1. Clear the old reminder.  
2. Create a new reminder.  
3. Use a near-term test time.  
4. Confirm that the reminder appears.  
  
## A category is missing or unclear  
## Check first  
1. Open **Categorize > All Categories**.  
2. Search for the category name.  
3. Confirm that it was not renamed.  
4. Confirm that you are viewing the correct mailbox.  
5. Use the category name rather than relying only on its color.  
## What to remember  
Access problems and sending permissions usually require support. Do not repeatedly change account settings to work around them.

## Page 66 — Resolve signature, automatic reply and Copilot issues  
## A signature does not appear  
## Check first  
1. Open a new message.  
2. Select **Signature**.  
3. Confirm that the signature exists.  
4. Open **Signatures**.  
5. Confirm the correct email account.  
6. Review the defaults for:  
    * New messages  
    * Replies and forwards  
7. Insert the signature manually to test it.  
## Try this  
* Reassign the default signature.  
* Save the signature again.  
* Restart Outlook.  
* Confirm that you are not comparing Classic Outlook with another platform’s signature settings.  
  
## Automatic replies did not start  
## Check first  
1. Open **File > Automatic Replies**.  
2. Confirm that **Send automatic replies** is selected.  
3. Check the start and end date.  
4. Check the start and end time.  
5. Review both internal and external tabs.  
6. Confirm that Outlook is using the correct work account.  
## Get help when  
* Automatic Replies is missing  
* Settings cannot be opened  
* Replies continue after the end time  
* A shared mailbox requires automatic-reply configuration  
  
## Copilot is missing  
## Check first  
1. Confirm that Outlook is using your licensed company account.  
2. Confirm that the message is in your primary Exchange Online mailbox.  
3. Confirm that you are using a supported Microsoft 365 version.  
4. Close and reopen Outlook.  
5. Check for Microsoft 365 updates when allowed.  
6. Confirm that connected experiences are enabled or managed by the organization.  
7. Try a supported message in your primary Inbox.  
## Copilot may not appear when  
* The message is in a shared mailbox  
* The message is in an archive mailbox  
* The account is unsupported  
* The mailbox is not hosted in Exchange Online  
* The message uses unsupported encryption  
* Organizational settings disable the feature  
## Get help when  
* Copilot is missing across supported messages  
* The license is assigned but the command never appears  
* Copilot repeatedly returns an error  
* Draft or summary features stop working after an update  
**Support path:** [Insert Copilot support path]  
  
## Copilot cannot summarize a message  
## Check first  
1. Confirm that the conversation is in your primary mailbox.  
2. Confirm that the thread contains multiple messages.  
3. Confirm that the email is not encrypted in an unsupported way.  
4. Close and reopen the message.  
5. Restart Outlook.  
6. Try another low-risk conversation.  
## Get help when  
* Supported threads consistently fail  
* The summary command appears but never completes  
* Copilot displays a repeated error message  
* The same issue affects drafting and coaching  
  
## Before contacting support  
Record:  
* Outlook version  
* Windows device name, when requested  
* Mailbox involved  
* Task you were completing  
* Exact error message  
* Date and time  
* Whether the issue occurs in Outlook on the web  
* Screenshot with sensitive information removed

## Page 69 — How other Outlook platforms differ  
## How other Outlook platforms differ  
This guide covers **Classic Outlook for Windows**.  
The separate guides for New Outlook, Outlook on the web and Outlook on iPadOS will use the same task order so they can later be combined into one master guide.  

| Task | New Outlook for Windows | Outlook on the web | Outlook on iPadOS |
| ------------------ | ------------------------------------ | -------------------------- | ------------------------------ |
| Interface | Web-based layout | Browser-based layout | Touch-first mobile layout |
| Rules | Settings-based workflow | Settings-based workflow | Limited management |
| Categories | Supported with different menus | Similar to New Outlook | Mobile command workflow |
| Shared mailboxes | Different account workflow | Open or add shared mailbox | Mobile mailbox workflow |
| Signatures | Separate New Outlook settings | Browser signature settings | Mobile signature settings |
| Automatic replies | Account settings | Browser account settings | Mobile settings |
| Calendar follow-up | Different email-to-calendar workflow | Browser workflow | Touch and keyboard workflow |
| Copilot summary | Supported where available | Supported where available | Availability and layout vary |
| Copilot Chat | Broader integration | Broader integration | Mobile capabilities vary |
| Attachments | Desktop and cloud options | Browser and cloud options | Files and mobile share sheet |
| Keyboard use | Windows shortcuts | Browser shortcuts | Apple Magic Keyboard shortcuts |
  
****Platform labels used in this series****  
## Same on most platforms  
The task follows a similar process, even when buttons move.  
## Different in New Outlook  
The workflow or setting location changes in New Outlook.  
## Different on the web  
The task uses browser settings or web menus.  
## Different on iPad  
The task uses touch controls, mobile menus or Apple Magic Keyboard shortcuts.  
## Not available  
The platform does not support the feature.  
## Availability may vary  
The feature depends on licensing, deployment, mailbox type or administrator configuration.  
## What to remember  
Use the guide that matches the Outlook version and device in front of you.

## Page 70 — Final reminder  
## Take control one decision at a time  
Before moving to the next message, ask:  
1. Does this need action from me?  
2. What is the next step?  
3. Where will I make it visible?  
4. When will I follow through?  
## Keep these habits  
* Respond now when the reply is quick.  
* Complete short tasks when practical.  
* Flag work you need to revisit.  
* Add reminders when timing matters.  
* Use categories to identify work clearly.  
* Reserve calendar time for focused work.  
* File or archive completed information.  
* Use rules only for predictable email.  
* Search before recreating work.  
* Confirm ownership in shared mailboxes.  
* Use Copilot when it helps.  
* Review every Copilot output.  
* Close the loop when the work is complete.
