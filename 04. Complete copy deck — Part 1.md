## Page 6 — Take control workflow  
## Take control one decision at a time  
Most inbox challenges are not only about volume. They come from not knowing what still needs attention.  
Use this workflow whenever you open an email:  
## Email → Decide → Make visible → Follow through  
## Start with one question  
**Does this need action from me?**  
If the answer is yes, ask:  
1. **What is the next step?**  
2. **Where will I make it visible?**  
## Four doors for every email  

| Door | Use it when | What to do |
| ---------------------- | ---------------------------------- | -------------------------------- |
| Respond now | A quick reply resolves the message | Send the reply |
| Do the task | The work can be completed quickly | Complete it, then file the email |
| Make follow-up visible | More work or waiting is required | Flag, categorize or calendar it |
| Let it go | No action is required | File, archive or delete it |
  
****What to remember****  
* **Visible beats remembered.**  
* Make the next action easy to find.  
* Use a simple system you will notice later.  
* Copilot can help you move faster.  
* You remain responsible for the final message.  
**Copilot starts the draft. You finish the message.**  
**Suggested visual**  
Create a horizontal process diagram:  
**Envelope → Signpost → Flag or calendar → Checkmark**

## Page 8 — Identify Classic Outlook  
## Identify Classic Outlook  
## Use this when  
Use this section when you are unsure which version of Outlook is open.  
## How to recognize Classic Outlook  
Classic Outlook normally includes:  
* A **File** tab in the upper-left corner  
* A traditional command **ribbon**  
* A folder pane on the left  
* A message list in the center  
* A reading pane on the right or bottom  
* Mail and Calendar navigation  
* A **Send/Receive** tab on the ribbon  
Microsoft identifies the presence of a Send/Receive tab as one way to distinguish Classic Outlook from New Outlook or Outlook on the web. (++[Microsoft Support](https://support.microsoft.com/en-us/office/import-or-export-a-set-of-rules-in-classic-outlook-f54b5bd2-40e0-426e-9f25-e51fa14eeb95?utm_source=chatgpt.com)++)  
## Confirm the version  
1. Open Outlook.  
2. Look at the ribbon across the top of the window.  
3. Confirm that you see the **File** tab.  
4. Look for the **Send/Receive** tab.  
5. Confirm that the screen resembles the example shown on this page.  
**Platform difference:** New Outlook and Outlook on the web use a web-based layout and do not provide every Classic Outlook command.

## Page 9 — Understand the Classic Outlook layout  
## Understand the Classic Outlook layout  
The layout may change slightly based on your view settings, but the main areas remain consistent.  

| Area | What it does |
| --------------- | ------------------------------------------------------------------ |
| Ribbon | Provides commands for the selected message, folder or Outlook item |
| Search box | Finds messages, people, contacts, tasks and calendar entries |
| Folder pane | Opens folders, favorites and shared mailboxes |
| Message list | Displays messages in the selected folder |
| Reading pane | Displays the selected message without opening a separate window |
| Navigation area | Switches between Mail, Calendar, People and Tasks |
| Status bar | Shows connection, item count, view and zoom information |
  
Outlook Search can locate messages, contacts, tasks and calendar entries from the Search box. (++[Microsoft Support](https://support.microsoft.com/en-us/office/how-to-search-in-outlook-d824d1e9-a255-4c8a-8553-276fb895a8da?utm_source=chatgpt.com)++)

## Page 10 — Customize the basic view  
## Customize the basic view  
## Use this when  
Adjust Outlook so messages and folders are easier to read and manage.  
## Change the Reading Pane  
1. Select the **View** tab.  
2. Select **Reading Pane**.  
3. Choose:  
    * **Right**  
    * **Bottom**  
    * **Off**  
4. Confirm the new layout works for your screen.  
## Show or hide the Folder Pane  
1. Select the **View** tab.  
2. Select **Folder Pane**.  
3. Choose the desired display option.  
4. Confirm that the folders you use remain easy to reach.  
## Change message preview  
1. Select the **View** tab.  
2. Select **Message Preview**.  
3. Choose how many preview lines appear.  
4. Review the message list.  
## Sort messages  
1. Open the folder you want to sort.  
2. Select the current sorting option above the message list.  
3. Choose a field such as:  
    * Date  
    * From  
    * Subject  
    * Categories  
    * Flag status  
4. Choose ascending or descending order when available.  
## Turn conversation view on or off  
1. Select the **View** tab.  
2. Select or clear **Show as Conversations**.  
3. Choose whether the change applies to:  
    * This folder  
    * All mailboxes  
## Reset a view  
1. Open the affected folder.  
2. Select the **View** tab.  
3. Select **Reset View**.  
4. Confirm the change.
