## Page 41 — Create an email signature  
## Create an email signature  
## Use this when  
Add approved contact information automatically to new messages, replies or forwards.  
## Before you start  
Confirm the approved:  
* Name and job title  
* Department or business area  
* Phone number  
* Email address  
* Company wording  
* Logo or image requirements  
* Legal or confidentiality language  
Use the current approved corporate standard.  
## Create a signature  
1. In Outlook, select **New Email**.  
2. In the new message window, select the **Message** tab.  
3. Select **Signature**.  
4. Select **Signatures**.  
5. Under **Select signature to edit**, select **New**.  
6. Enter a short name for the signature.  
7. Select **OK**.  
8. In the **Edit signature** area, enter the approved signature content.  
9. Apply approved formatting.  
10. Review spacing, links and alignment.  
11. Select **OK** to save the signature.  
12. Close the unfinished message when it is no longer needed.  
## Suggested signature structure  
**[Employee name]** [Job title] [Department or business area] [Approved phone number] [Approved email or website information]

## Page 42 — Set, insert and maintain signatures  
## Set a default signature  
## Use this when  
Choose which signature Outlook should add automatically.  
1. Open a new message.  
2. Select **Signature**.  
3. Select **Signatures**.  
4. Under **Choose default signature**, select the correct email account.  
5. Under **New messages**, select the signature Outlook should add automatically.  
6. Under **Replies/forwards**, select:  
    * The signature to add automatically  
    * **None** when no automatic signature is required  
7. Select **OK**.  
## Insert a signature manually  
1. Create a new message, reply or forward.  
2. Select the **Message** tab.  
3. Select **Signature**.  
4. Choose the signature from the list.  
5. Confirm that it appears in the message.  
## Change the signature in one message  
1. Select the existing signature text.  
2. Remove it when needed.  
3. Select **Signature**.  
4. Choose a different approved signature.  
## Edit an existing signature  
1. Open **Signature > Signatures**.  
2. Select the signature under **Select signature to edit**.  
3. Update the content.  
4. Review the formatting.  
5. Select **OK**.  
## Delete an outdated signature  
1. Open **Signature > Signatures**.  
2. Select the outdated signature.  
3. Select **Delete**.  
4. Confirm the deletion.  
5. Review the default settings so they do not point to the deleted signature.  
6. Select **OK**.  
## Signature review checklist  
Check the signature when:  
* Your title changes  
* Your team changes  
* Your phone number changes  
* Corporate branding changes  
* Required wording changes  
* A duplicate or outdated signature appears  
## What to remember  
Classic Outlook signatures may be managed separately from signatures used in New Outlook, Outlook on the web or Outlook on iPadOS.  
## Avoid this  
* Do not assume a signature created on one platform appears automatically on every platform.  
* Do not leave several outdated versions in the signature list.  
* Do not use different contact information across approved signatures.  
* Do not manually alter required corporate wording.  
## Platform difference  
Signature creation and default settings use different menus in New Outlook, Outlook on the web and Outlook on iPadOS.

## Page 43 — Turn on automatic replies  
## Set automatic replies  
## Use this when  
Let senders know that you are unavailable, responses may be delayed or another contact should be used.  
## Before you start  
Decide:  
* Start date and time  
* End date and time  
* Return date  
* Alternate contact  
* Urgent support path  
* Whether external senders need a separate message  
## Turn automatic replies on  
1. Select **File**.  
2. Select **Automatic Replies**.  
3. Select **Send automatic replies**.  
4. To control when the reply runs, select **Only send during this time range**.  
5. Enter the start date and time.  
6. Enter the end date and time.  
7. Select the **Inside My Organization** tab.  
8. Enter the message for coworkers.  
9. Review the wording.  
10. Select **OK**.  
If no time range is selected, the automatic reply remains active until it is turned off manually.  
## Suggested internal message  
Thank you for your message. I am away from the office and will return on [date].  
For urgent assistance, contact [name, team or support path]. Otherwise, I will respond after I return.

## Page 44 — Manage external replies and turn them off  
## Send a reply outside the organization  
## Use this when  
External senders need to know that your response may be delayed.  
1. Open **File > Automatic Replies**.  
2. Select the **Outside My Organization** tab.  
3. Select **Auto-reply to people outside my organization** when appropriate.  
4. Choose the available audience option:  
    * My contacts only  
    * Anyone outside my organization  
5. Enter the external reply.  
6. Remove internal-only details.  
7. Review the alternate contact and support path.  
8. Select **OK**.  
## Suggested external message  
Thank you for your message. I am currently unavailable and will return on [date].  
For immediate assistance, contact [approved team or contact]. Otherwise, I will respond after I return.  
## Use care with external replies  
An external automatic reply may be sent to:  
* Customers  
* Producers  
* Vendors  
* Newsletters  
* Automated systems  
* Unwanted messages  
Include only information suitable for a broad external audience.  
## Update automatic replies  
1. Select **File**.  
2. Select **Automatic Replies**.  
3. Change the dates, times or message.  
4. Review both internal and external tabs.  
5. Select **OK**.  
## Turn automatic replies off  
When automatic replies are active, Outlook may display a notice below the ribbon.  
To turn them off:  
1. Select **Turn off** in the notice.  
Or:  
1. Select **File**.  
2. Select **Automatic Replies**.  
3. Select **Do not send automatic replies**.  
4. Select **OK**.  
## Troubleshooting: Automatic Replies is missing  
If the command is unavailable:  
1. Confirm that you are using the company Microsoft 365 work account.  
2. Confirm that Classic Outlook is connected.  
3. Restart Outlook.  
4. Check whether the correct account is selected.  
5. Use [insert support path] if the option remains unavailable.  
Do not create a rule-based automatic reply unless that approach is approved for the account and support process.  
## Avoid this  
* Do not include unnecessary travel details.  
* Do not include customer, member or confidential information.  
* Do not name an alternate contact without confirming availability.  
* Do not leave an external reply active longer than needed.  
* Do not use identical internal and external messages without reviewing the content.  
## Platform difference  
Automatic-reply settings use different menus in New Outlook, Outlook on the web and Outlook on iPadOS.

## Page 46 — Locate and open a shared mailbox  
## Locate a shared mailbox  
## Use this when  
A team or department uses one mailbox that several employees monitor and respond from.  
## Before you start  
You must be granted access before the shared mailbox can be used.  
Access may include:  
* Read and manage  
* Send As  
* Send on Behalf  
The required permissions depend on the work you need to complete.  
## Locate an automatically added mailbox  
After access is granted:  
1. Close Outlook.  
2. Restart Outlook.  
3. Look in the Folder Pane.  
4. Locate the shared mailbox below your primary mailbox.  
5. Select the arrow beside the mailbox name to expand it.  
6. Open **Inbox**.  
7. Review the available folders.  
The mailbox may take time to appear after access is granted.  
## If the mailbox does not appear  
1. Wait several minutes after access is approved.  
2. Close Outlook completely.  
3. Restart Outlook.  
4. Confirm that the correct Outlook profile is open.  
5. Contact [insert shared mailbox support path] if the mailbox remains unavailable.

## Page 47 — Add a shared mailbox manually  
## Add a shared mailbox manually  
## Use this when  
Access has been approved, but the mailbox does not appear automatically.  
## Add the mailbox to Classic Outlook  
1. Select **File**.  
2. Select **Account Settings**.  
3. Select **Account Settings** again.  
4. On the **Email** tab, select your company work account.  
5. Select **Change**.  
6. Select **More Settings**.  
7. Open the **Advanced** tab.  
8. Under the mailbox list, select **Add**.  
9. Enter the shared mailbox name or email address.  
10. Select **OK**.  
11. Select **Apply** when available.  
12. Select **OK**.  
13. Select **Next** or complete the remaining account window.  
14. Select **Finish**.  
15. Close Account Settings.  
16. Restart Outlook.  
17. Confirm that the mailbox appears in the Folder Pane.

## Page 48 — Send from a shared mailbox  
## Send from a shared mailbox  
## Use this when  
The recipient should see the approved team or departmental address instead of your individual address.  
## Show the From field  
1. Select **New Email**.  
2. Open the **Options** tab.  
3. In **Show Fields**, select **From**.  
4. Confirm that the From field appears above the recipient fields.  
## Select the shared address  
1. Select **From**.  
2. Choose the shared mailbox from the list.  
If it is not listed:  
1. Select **Other Email Address**.  
2. Enter the shared mailbox address.  
3. Select the address.  
4. Confirm that it appears in the From field.  
## Complete and send the message  
1. Add the recipients.  
2. Enter a clear subject.  
3. Write the message.  
4. Add the approved signature if needed.  
5. Review the From address.  
6. Review recipients, attachments and links.  
7. Select **Send**.  
The shared address should remain available in the From list after it has been used successfully.

## What happens next  
Depending on the assigned permission:  
* **Send As:** The message appears to come directly from the shared mailbox.  
* **Send on Behalf:** The message may show that you sent it on behalf of the shared mailbox.  
## What to remember  
Check the From address before every shared-mailbox message.  
## Avoid this  
* Do not send from your personal address when the shared address is required.  
* Do not send from the shared address for unrelated work.  
* Do not assume Sent Items will be stored where your team expects.  
* Do not use an unapproved signature or closing.  
* Do not send when another team member is already handling the message.

## Page 49 — Reply, search and organize shared messages  
## Reply from the correct mailbox  
1. Open the shared mailbox.  
2. Open the message.  
3. Select **Reply** or **Reply All**.  
4. Review the **From** field.  
5. Confirm that the shared mailbox is selected.  
6. Write the response.  
7. Check the conversation for another team response.  
8. Review the final message.  
9. Select **Send**.  
If the From field shows your individual address:  
1. Select **From**.  
2. Choose the shared mailbox.  
3. Continue the reply.  
## Search a shared mailbox  
1. Select a folder inside the shared mailbox.  
2. Select the Search box.  
3. Confirm the search scope.  
4. Enter the search terms.  
5. Press **Enter**.  
6. Refine by sender, subject, date or attachments when needed.  
7. Confirm that the result belongs to the shared mailbox.  
## Organize shared messages  
Follow the approved team process for:  
* Flags  
* Categories  
* Folder names  
* Assignment or ownership  
* Completed messages  
* Waiting items  
* Sent-message storage  
* Duplicate-response prevention  
## Suggested shared-workflow statuses  

| Status        | Meaning                                 |
| ------------- | --------------------------------------- |
| New           | Not yet reviewed                        |
| In progress   | A team member is handling it            |
| Waiting       | Waiting for information or response     |
| Ready to send | Work is complete and needs final review |
| Complete      | Response sent or work closed            |
  
Use the team’s approved system when one already exists.  
## Prevent duplicate responses  
Before replying:  
1. Review the conversation.  
2. Check shared Sent Items when available.  
3. Check flags, categories or team status.  
4. Confirm whether another employee owns the work.  
5. Add or update the agreed status.  
6. Send only when ownership is clear.
