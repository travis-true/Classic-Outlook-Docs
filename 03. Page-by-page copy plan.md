## Page 6 — Take control workflow  
**Page purpose:** Connect the guide to the live session and establish the decision framework.  
**Heading:** Take control one decision at a time  
**Core workflow:**  
**Email → Decide → Make visible → Follow through**  
**Primary question:**  
Does this need action from me?  
**If yes, ask:**  
1. What is the next step?  
2. Where will I make it visible?  
**Four doors table:**  

| Door                   | What to do                      |
| ---------------------- | ------------------------------- |
| Respond now            | Send a quick reply              |
| Do the task            | Complete it if it is fast       |
| Make follow-up visible | Flag, categorize or calendar it |
| Let it go              | File, archive or delete it      |
  
**Key reminders:**  
* Visible beats remembered.  
* A simple system you use is better than a perfect system you ignore.  
* Copilot starts the draft. You finish the message.

## Pages 12–13 — Read, reply, reply all and forward  
**Task title:** Read and respond to an email  
**Use this when:** Open a message, respond to the sender or share the message with someone else.  
**Page 12 procedures:**  
**Open a message**  
1. Select the message in the message list.  
2. Read it in the Reading Pane or double-click to open it.  
3. Review the sender, recipients, subject and attachments.  
**Reply**  
1. Select Reply.  
2. Write your response.  
3. Review the recipient.  
4. Select Send.  
**Reply all**  
1. Select Reply All.  
2. Confirm that everyone needs the response.  
3. Remove unnecessary recipients.  
4. Write the response and select Send.  
**Screenshot placeholder:**  
[Insert screenshot here: Open email with Reply, Reply All and Forward commands highlighted.]  
**Alt text:** Open email in Classic Outlook showing the Reply, Reply All and Forward commands.  
**Page 13 procedures:**  
**Forward**  
1. Select Forward.  
2. Add the new recipient.  
3. Add context above the forwarded message.  
4. Review attachments.  
5. Select Send.  
**Pop out a response**  
1. Start the reply.  
2. Select Pop Out.  
3. Complete the message in the separate window.  
**Decision table:**  

| Use       | When                                    |
| --------- | --------------------------------------- |
| Reply     | Only the sender needs your response     |
| Reply All | Everyone included needs the response    |
| Forward   | A new person needs the original context |
  
**Avoid this:**  
Do not use Reply All when only one person needs the response.  
**Platform difference callout:** Command locations differ on New Outlook, web and iPadOS, but the decision between Reply, Reply All and Forward stays the same.
