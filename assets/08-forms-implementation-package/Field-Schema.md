# Field schema

Status: Production draft

## Guide feedback form

| Order | Question | Type | Required | Choices / branching | Notes |
|---:|---|---|---|---|---|
| 1 | Which resource are you reviewing? | Choice | Yes | Classic Outlook detailed guide; Classic Outlook task map; Which Outlook am I using? card; Shared mailbox mini-QRG; Copilot troubleshooting card; Search syntax cheat sheet; Keyboard-shortcut appendix; Short demonstration video; Other | Purpose: identify resource. |
| 2 | Which task or section did you use? | Short answer | No | None | Do not include sensitive information. |
| 3 | Were you able to complete the task? | Choice | Yes | Yes, without help; Yes, with some help; No; I was only reviewing the information | Use for success trend. |
| 4 | How clear were the instructions? | Rating | Yes | 1 to 5; Not clear / Very clear | Include labels. |
| 5 | How useful were the screenshots or visuals? | Rating | No | 1 to 5; Not useful / Very useful | Include Not applicable where supported. |
| 6 | What was most helpful? | Long answer | No | None | Sensitive-information warning. |
| 7 | What was confusing or missing? | Long answer | No | None | Sensitive-information warning. |
| 8 | What should be improved? | Long answer | No | None | Sensitive-information warning. |
| 9 | Which Outlook platform were you using? | Choice | Yes | Classic Outlook for Windows; New Outlook for Windows; Outlook on the web; Outlook on iPadOS; Not sure | Supports platform-specific triage. |
| 10 | May IT Training & Enablement contact you for follow-up? | Choice | Yes | Yes; No | Branch to contact information when Yes. |
| 11 | Contact information | Short answer | Conditional | Show only when Q10 = Yes | Use work contact only. |

Confirmation: Thank you for helping us improve these resources. Your feedback will be reviewed by IT Training & Enablement.

## Report outdated instructions form

| Order | Question | Type | Required | Choices / branching | Notes |
|---:|---|---|---|---|---|
| 1 | Which resource appears outdated? | Choice | Yes | Classic Outlook detailed guide; Classic Outlook task map; Which Outlook am I using? card; Shared mailbox mini-QRG; Copilot troubleshooting card; Search syntax cheat sheet; Keyboard-shortcut appendix; Short demonstration video; SharePoint resource page; Other | Identify resource. |
| 2 | What is the page, section, video, or task name? | Short answer | Yes | None | Use title, page, or scene name. |
| 3 | What type of issue did you find? | Choice | Yes | Step does not work; Button or menu moved; Screenshot does not match; Feature is missing; Wording is unclear; Link is broken; Support contact is incorrect; Accessibility issue; Platform mismatch; Other | Drives triage. |
| 4 | What did the instructions tell you to do? | Long answer | Yes | None | No sensitive information. |
| 5 | What happened instead? | Long answer | Yes | None | No sensitive information. |
| 6 | Which Outlook platform were you using? | Choice | Yes | Classic Outlook for Windows; New Outlook for Windows; Outlook on the web; Outlook on iPadOS; Not sure | Supports validation. |
| 7 | What device were you using? | Choice | Yes | Windows laptop; iPad; Other | Supports device triage. |
| 8 | What date did you notice the issue? | Date | Yes | None | Use local date. |
| 9 | Upload a screenshot | File upload | No | Only if approved | Remove or blur sensitive details. |
| 10 | How urgent is the issue? | Choice | Yes | Critical — The instructions may create risk or data loss; High — The task cannot be completed; Medium — The task is confusing or partially incorrect; Low — Minor wording, visual, or link issue | Pair color with text labels. |
| 11 | Additional details | Long answer | No | None | No sensitive information. |

Confirmation: Thank you. IT Training & Enablement will review the reported issue and update the resource when needed.
