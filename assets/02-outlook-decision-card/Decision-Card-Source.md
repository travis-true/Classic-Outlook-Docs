# Which Outlook am I using?

Status: Production draft

Use the guide that matches the application or device in front of you.

## Decision path

1. Are you using an iPad?
   - Yes: You are using Outlook on iPadOS. Use the Outlook on iPadOS guide.
   - No: Continue.
2. Is Outlook open inside Chrome or Edge?
   - Yes: You are using Outlook on the web. Use the Outlook on the web guide.
   - No: Continue.
3. Do you see a Settings button in the upper-right area of Outlook?
   - Yes: You are using New Outlook for Windows. Use the New Outlook for Windows guide.
   - No: You are probably using Classic Outlook for Windows. Use the Classic Outlook for Windows guide.

## What to look for

### Classic Outlook for Windows
- File tab.
- Traditional ribbon.
- Send/Receive tab.
- Folder Pane, Message List and Reading Pane.

### New Outlook for Windows
- Modern web-style layout.
- Settings button in the upper-right area.
- Simplified toolbar.
- No traditional Send/Receive tab.

### Outlook on the web
- Outlook inside a browser tab.
- Browser address bar above Outlook.
- Settings gear near the upper-right area.
- No separate desktop Outlook window.

### Outlook on iPadOS
- Outlook mobile app.
- Touch-first layout.
- Account icon or profile menu.
- Apple Magic Keyboard and trackpad support.
