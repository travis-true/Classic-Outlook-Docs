# Shared mailboxes in Classic Outlook

Status: Production draft

## High-level overview
Use this two-page quick reference guide when you have approved access to a shared mailbox in Classic Outlook for Windows. A shared mailbox is opened through your own work account. Do not sign in directly with the shared mailbox address.

## Architectural diagram concepts
Your work account is the entry point. Permissions control what you can do in the shared mailbox: read and manage, Send As, and Send on Behalf. Classic Outlook shows the shared mailbox in the Folder Pane after access is granted or after it is added manually. Sent-item location can vary by Exchange configuration, so teams must confirm where sent messages are expected to appear.

## Step-by-step setup
### Permission prerequisites
- Approved mailbox access is required before employee steps will work.
- Access may include read and manage, Send As, or Send on Behalf.
- Use [[APPROVED SHARED-MAILBOX ACCESS PATH REQUIRED]] to request or check access.

### Open the shared mailbox
If it appears automatically:
1. Restart Classic Outlook after access is granted.
2. Find the mailbox in the Folder Pane.
3. Select the arrow beside its name.
4. Open Inbox.

If it does not appear:
1. Select File.
2. Select Account Settings.
3. Select Account Settings again.
4. Select your work account.
5. Select Change.
6. Select More Settings.
7. Open Advanced.
8. Select Add.
9. Enter the shared mailbox address.
10. Complete the prompts and restart Outlook.

### Send from the shared mailbox
1. Select New Email.
2. Open Options.
3. Select From.
4. Select the shared mailbox address.
5. Add recipients and a subject.
6. Write the message.
7. Confirm ownership and the From address before sending.
8. Review and select Send.

If the mailbox is not listed, select From, select Other Email Address, and enter the shared mailbox address.

### Reply from the shared mailbox
1. Open the message in the shared mailbox.
2. Select Reply or Reply All.
3. Check the From field.
4. Select the shared address if needed.
5. Review the full conversation.
6. Confirm no one else has replied.
7. Write and send the response.

### Search the mailbox
1. Open a folder inside the shared mailbox.
2. Select the Search box.
3. Confirm the search scope.
4. Enter a sender, subject, or keyword.
5. Add filters when needed.
6. Confirm the result belongs to the shared mailbox.

## Common Gotchas
- Do not share mailbox passwords.
- Do not use the shared address for unrelated work.
- Do not assume unread means unassigned.
- Do not imply sent-item behavior is universal; confirm team expectations.
- Do not use Copilot inside the shared mailbox and assume it can access mailbox content.
