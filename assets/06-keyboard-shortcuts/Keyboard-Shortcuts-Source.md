# Classic Outlook keyboard shortcuts

Status: Production draft  
Audience: Windows laptop users using Classic Outlook for Windows with company work accounts.

## High-level overview
Use shortcuts when they make Outlook faster and comfortable for you. You do not need to memorize every key. Start with five shortcuts, then add more when they help your work.

## Architectural diagram concepts
The appendix groups shortcuts by the work pattern employees use most: create, respond, navigate, search, organize, calendar and format. Each table includes the action, the shortcut, and a plain-language context note so employees know where the shortcut works.

## Step-by-step setup
1. Confirm you are using Classic Outlook for Windows.
2. Pick one section that matches the task you do most.
3. Practice the “Start with five” shortcuts for one week.
4. Add context-dependent shortcuts only when the related pane, message or calendar item is selected.
5. Ask for help through [[APPROVED SUPPORT PATH REQUIRED]] if a shortcut does not work on your device.

## Start with five
* Ctrl+Shift+M — New email
* Ctrl+R — Reply
* Ctrl+Shift+R — Reply All
* Ctrl+E — Search
* Ctrl+Shift+V — Move to folder

## Common Gotchas
* Shortcuts can vary by Outlook version, keyboard layout and organization configuration.
* Some shortcuts work only when the correct pane or item is selected.
* Browser shortcuts may conflict with Outlook shortcuts in Outlook on the web.
* Permanent delete bypasses normal delete behavior. Use it only when appropriate and approved.
* The iPadOS guide will include Apple Magic Keyboard shortcuts separately.
