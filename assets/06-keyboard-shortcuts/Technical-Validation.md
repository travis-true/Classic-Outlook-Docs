# Technical validation — Keyboard-shortcut appendix

Status: Production draft

All included shortcuts were checked against official Microsoft Support keyboard shortcut references on June 27, 2026. Tenant-specific validation is still required before publication because Microsoft 365 update channels, keyboard layouts, and organization policies may affect behavior.

Primary source: Microsoft Support, “Keyboard shortcuts for Outlook,” https://support.microsoft.com/en-us/accessibility/outlook/keyboard-shortcuts-for-outlook

Secondary source for mobile boundary note: Microsoft Support, “What keyboard shortcuts are available?” https://support.microsoft.com/en-us/outlook/what-keyboard-shortcuts-are-available

Removed or softened items from the source package when current support wording or configuration dependence required caution:

* Category shortcuts are described as context-dependent and possibly configured by the organization.
* Archive with Backspace is marked configuration-dependent.
* Mark complete is described through Follow Up or configured Quick Steps instead of as a universal shortcut.
* Permanent delete is identified as a warning.
