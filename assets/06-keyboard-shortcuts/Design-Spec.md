# Design spec — Classic Outlook keyboard shortcuts

Status: Production draft

## Format
* US Letter portrait.
* Target length: three pages; four pages allowed for accessible spacing.
* Editable Word document created with python-docx.
* No screenshots or mockups.

## Brand
* Use the official BCBSKS primary logo file at 2 inches wide or larger.
* Use Blue #005EB8, Midnight #002855, Bluejay #41B6E6, approved grays, and white.
* Warning red #C63527 is reserved for the permanent-delete warning.
* Include the licensee disclosure.

## Typography
* Title: Georgia Bold, 20 pt.
* Section headings: Georgia Bold, 13–15 pt.
* Body: Arial, 11 pt.
* Table text: Arial, 10 pt minimum.
* Left aligned. No justified body copy.

## Accessibility
* Real Word heading styles.
* Real Word tables with repeated header rows where supported.
* Real bullets and numbered lists.
* No essential content in floating text boxes.
* No color-only meaning.
* Plain Talk wording.
* Document properties and English (United States).
