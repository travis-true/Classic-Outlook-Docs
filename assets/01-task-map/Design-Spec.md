# Classic Outlook task map design spec

Status: Production draft

## High-level overview

Create `Take-Control-of-Your-Inbox_Classic-Outlook_Task-Map_v1.0.docx` as a one-page, US Letter, portrait, editable Word document. Use real text, Word styles, real tables, and inline logo art. Do not use screenshots.

## Architectural diagram concepts

- Header: BCBSKS logo, title, subtitle, production draft label.
- Workflow banner: Email → Decide → Make visible → Follow through.
- Body: Six panels in a two-column table grid.
- Footer: disclosure, placeholders, guide name, version.

## Accessibility notes

- Body text minimum: 11 pt.
- Use real heading styles and real Word tables.
- Do not depend on color alone.
- Use descriptive placeholders and readable labels.
- Use English (United States) document metadata.

## Common Gotchas

- The one-page target still requires human Word render review.
- The logo must not be cropped, stretched, recolored, or flattened into the page design.
- Do not label the asset final or accessibility-compliant before human review.
